**목적**: IFRS 연결 재무제표를 재구성하여, 자본조달 → 자산투자 → 수익률 분해 → 이익귀속 분석을 수행하는 기업 분석 대시보드

**레이아웃**: 각 Section은 3단 구조

- 중앙 (H~L열): 계산식/시그널/보조 지표
- 우측 (N열): 해석 가이드

## Section 4. 3대 수익률 분해

**핵심 질문**: "각 자산이 얼마를 벌어오는가?"

### 수익률 항목 (3개년 + 가중평균)
| 지표 | 공식 | 의미 |
|------|------|------|
| 영업자산이익률 | 영업이익 / 평균영업자산 | 본업 자산 효율성 (핵심) |
| 비영업자산이익률 | (당기순이익-영업이익+이자비용+법인세) / 평균비영업자산 | 유휴자산 수익률 |
| 차입이자율 | 이자비용 / 평균외부차입 | 실효 조달비용 |
| 지배주주 ROE | 지배주주순이익 / 평균지배주주지분 | 최종 주주수익률 |
| Spread (ROE-Ke) | ROE - 요구수익률 | 양수=초과이익, 음수=가치파괴 |

### ROE 분해 공식
```
ROE = 영업자산이익률 x 영업자산비율
    + 비영업자산이익률 x (1 - 영업자산비율)
    - 차입이자율 x (외부차입/자기자본)
    위 결과에 세율, 비지배비율 적용
```

### 가중평균 산출
- 공식: (최근 x 3 + 중간 x 2 + 과거 x 1) / 6
- 우선순위: 1) FnGuide 컨센서스 ROE, 2) 3개년 가중평균, 3) 최근 4분기 실적

## Section 5. 이익 귀속 흐름 (Profit Waterfall)

**핵심 질문**: "벌어들인 이익이 누구에게 가는가?"

### Waterfall 구조
```
① 영업이익     = 영업자산 x 영업자산이익률     (~90% 비중)
② 비영업이익   = 비영업자산 x 비영업이익률       (~10%)
   총이익       = ① + ②                          (100%)
③ (-) 이자비용  = 외부차입 x 차입이자율
④ (-) 법인세    = (총이익-이자비용) x 유효세율
⑤ (-) 비지배분  = 당기순이익 x 비지배귀속비율
★ 지배주주순이익 = 당기순이익 - 비지배분
```

### 핵심 체크
- 영업이익 비중 90%+ = 본업형 기업
- 비영업이익 비중 높으면 이익의 질 의심
- 비지배 귀속 비율 높으면 내 몫 줄어듦

## Section 6. 시계열 진단 (Trend Signals)

**핵심 질문**: "어디로 가고 있는가?"

### 관찰 포인트
| 포인트 | 시그널 |
|--------|--------|
| 영업자산이익률 추세 | 핵심 드라이버. 올라가면 ROE 전체가 따라감 |
| 영업자산 규모 | 확대 + 이익률 유지 = 이익 레벨업 |
| 외부차입/자기자본 | 감소 추세 = 건전한 성장 |
| 영업자산비율 | 상승 = 유휴 현금이 투자로 전환 |
| 영업부채 | 증가 = 무이자 자본(공짜 자본) 증가 |
| Spread (ROE-Ke) | 음(-) 전환 시 S-RIM 적정가 < 장부가 |

### 시그널 해석
- 정배열 (ROE > Ke): 초과이익 창출 중, Spread 확대 추세면 매수 적기
- 역배열 (ROE < Ke): 가치 파괴 중, 턴어라운드 신호까지 관망/회피

| 지표 | 분자 | 분모 |
|------|------|------|

### E. 이익 Waterfall 구성 로직 (예상 손익 역산)
```
Step 1: 예상 영업이익     = 영업자산(최근) x 영업자산이익률(선택)
Step 2: 예상 비영업이익   = 비영업자산(최근) x 비영업이익률(선택)
Step 3: 예상 이자비용     = 외부차입(최근) x 차입이자율(선택)
Step 4: 예상 법인세       = (Step1 + Step2 - Step3) x 유효세율
Step 5: 예상 당기순이익   = Step1 + Step2 - Step3 - Step4
Step 6: 예상 지배주주순이익 = Step5 x 지배주주귀속비율
Step 7: 지배주주귀속비율  = 직전 지배주주순이익 / (지배+비지배)
Step 8: 유효세율          = 직전 법인세 / 세전이익 (기본값 22%)
```

### F. 수익률 선택 우선순위
1. **1순위**: FnGuide 컨센서스 ROE
2. **2순위**: 최근 3개년 가중평균 = (최근x3 + 중간x2 + 과거x1) / 6
3. **3순위**: 최근 4분기 실적
4. **별도**: 사용자 직접 입력 (시나리오 분석)
